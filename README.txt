Criei um banco de dados com Python usando diconário
Criei também algumas funções para manipulação desses dados.


incluir() - Para incluir um novo cadastro
excluir_por_nome() - Para excluir um cadastro
alterar_dados() - Para alterar uma indormação
listarBD() - Retorna todos os nomes e CPFs cadastrados por ordem alfabética
consulta_completa_nome() - Retorna todas as informações de um cadastro pesquisando pelo nome
consulta_completa_cpf() - Retorna todas as informações de um cadastro pesquisando pelo CPF
